Foundation

Grid

INTRO

BetterWorks app is built on a vertical 12-column grid system with fixed gutters at 20px. The total width for the layout grids are set for 960px.

Column widths are defined as percentages and therefore are responsive to various viewports. For a better experience on a wide range of devices, breakpoints are set to adjust the layout based on the grid system. To be precise, the border and padding is included in the width of the grid columns according to border-box.

LAYOUT TYPE

We apply our grid system in three types of layouts: Centered Fixed Container, Full Fluid Container and Fluid Container with Fixed Navigation.
