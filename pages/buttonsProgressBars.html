Buttons & Progress Bars

Use any of the available button classes to quickly create a styled button. Fancy larger or smaller buttons? Add .btn-lg or .btn-sm for additional sizes.



Primary Buttons

Primary buttons are used for the highest ranking actions per page. They emphasize they key elements a user would manipulate for navigation.

Small

Normal

Large


<button type="button" class="btn btn-primary"></button>
<button type="button" class="btn btn-primary" disabled="disabled"></button>

Success Buttons

Success buttons are used in multi-step forms (i.e. goal creation). They indicate a finish line. They should be the last button a user clicks in a multi-step process.

Small

Normal

Large


<button type="button" class="btn btn-success"></button>
<button type="button" class="btn btn-success" disabled="disabled"></button>

Alternative Primary Buttons

Alternative primary buttons are used when primary buttons cannot. This would occur when a button needs to be placed on an already highly saturated, colorful background. Both primary and secondary buttons are styled the same on saturated backgrounds.

Small

Normal

Large

<button type="button" class="btn btn-primary-alt"></button>
<button type="button" class="btn btn-primary-alt" disabled="disabled"></button>

Secondary Buttons

Secondary buttons are used for secondary actions. When paired with a primary button, the secondary button will always be on the left.

Small

  • Action
  • Another action
  • Something else here
  • Separated link

Normal

  • Action
  • Another action
  • Something else here
  • Separated link

Large

  • Action
  • Another action
  • Something else here
  • Separated link


<!-- Dropdown Secondary Button -->
<div type="button" class="btn-group">
  <button type="button" class="btn btn-default dropdown-toggle" data-toggle="dropdown" aria-expanded="false">
    Dropdown <span class="caret"></span>
  </button>
  <ul class="dropdown-menu" role="menu">
    <li><a href="#">Action</a></li>
    <li><a href="#">Another Action</a></li>
    <li><class="divider"></li>
    <li><a href="#">Separated Link</a></li>
  </ul>
</div>

Progress Bars

Progress bars are used for displaying the progress of goals. When a goal is on track, the progress bar is green. When a goal is 25% behind schedule, the progress bar is yellow. When a goal is 50% behind schedule, the progress bar is red.

51%

25%
51%

70%

Progress pies are used for displaying the progress of goals, just like progress bars. When a goal is on track, the progress pie is green. When a goal is 25% behind schedule, the progress pie is yellow. When a goal is 50% behind schedule, the progress pie is red.
