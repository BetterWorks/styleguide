Typography

BetterWorks uses the District typeface for it's web application. The global default font-size is 14px, with a line-height of 1.5. This is applied to the <body> and all paragraphs.



Heading 1, 2

This is the product's main heading style. It is used to clearly define primary sections of the product. It can never be used inline. It will always appear above subheaders.


This is an H1, H2 Example


<h1></h1>
<h2></h2>

<style>
h1,
h2 {
  color: #333;
  font-family: District-Book;
  font-size: 22px;
}
</style>

Heading 3, 4

This heading style is used for subheads, usually right below primary headers (H1,H2). They give additional context into a subsection of the product. It can never be used inline.


This is an H3, H4 Example


<h3></h3>
<h4></h4>

<style>
h3,
h4 {
  color: #999;
  font-family: District-Light;
  font-size: 18px;
}
</style>

Heading 5

The H5 acts as a body title. It can be used both inline as well as block. Use it sparingly. Don't go crazy and use it to bold every keyword. If there's a lot of body content, these are great. They help guide the user's eye and define a clear hierarchy.


This is an H5 Example

<h5></h5>

<style>
h5,
strong {
  color: #333;
  font-family: District-Medium;
  font-size: 14px;
}
</style>

Text Links

Subheaders (H3,H4) should never be links. If they're H5, H1, or H2 they adopt the parent formatting, i.e. H5 links can be both inline and block etc.


This is a link example. 


<a href="#"></a>

<style>
a {
  color: #5bc0de;
  font-family: District-Light;
  text-decoration: none;
}
</style>

Ghost Text

Ghost text is deemphasized body text. It grants greater detail and clarity without overtaking the body text. Use it sparingly. It can be used inline or block.


This is a ghost text example...spooooooky! 


<p class="ghost"></p>

<style>
.ghost {
  color: #999;
  font-family: District-Light;
  font-size: 14px;
}
</style>

Detail Text

Use detail text only when absolutely necessary. Because of its size it can easily be missed. Its position on the page should be chosen very carefully. Think of it as ghost text's little brother.


This is a detail text example. 


<p class="detail"></p>

<style>
.detail {
  color: #999;
  font-family: District-Light;
  font-size: 12px;
}
</style>

Placeholder Text

Placeholder text should only be used with interactive, form elements. It informs the user what to input and provides better usability than just labels alone. 



<input placeholder="Placeholder Example">		   		 
<style>
.form-control::-webkit-input-placeholder {
   color: #999;
   font-family: 'District-LightItalic';
   font-size: 14px;
}
</style>
