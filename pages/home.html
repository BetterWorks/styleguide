Styleguide

Welcome to the BetterWorks Styleguide! This guide was created to streamline the development of BetterWorks' products. It centralizes patterns to help clearly communicate brand standards between departments. It includes both visual examples as well as their font-end, HTML & CSS counterparts adhering to current web standards.
