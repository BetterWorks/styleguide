    Grid System

    The 960 Grid System is an effort to streamline web development workflow by providing commonly used dimensions, based on a width of 960 pixels.


    BetterWorks Grid

    BetterWorks uses the 15 column variant, divided into portions that are 44 pixels wide. Each column has 10 pixels of margin on the left and right, which create 20 pixel wide gutters between columns.
